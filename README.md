# Mi Primer Proyecto con Claude Code

Un repositorio de prueba para aprender a colaborar con Claude Code.

## Descripción

Este proyecto fue creado como primer ejercicio de flujo Git + GitHub usando Claude Code.

## Uso

Clona el repositorio y explora los archivos:

```bash
git clone https://github.com/Andreshab/mi-primer-pr-con-claude.git
cd mi-primer-pr-con-claude
```
